# Kalkulator Sederhana
Kalkulator berbasis HTML, CSS, dan JavaScript dengan fitur lengkap seperti operasi dasar, memori, riwayat perhitungan, dan antarmuka modern.

## Fitur Utama
**1. Operasi Matematika Dasar**
Penjumlahan, pengurangan, perkalian, dan pembagian.

**2. Fitur Memori (MC, MR, MS, M+, M-)**
Menyimpan, mengambil, menambah, dan mengurangi nilai memori kalkulator.

**3. Riwayat Perhitungan**
Menampilkan 5 riwayat terakhir dan dapat dihapus satu per satu atau sekaligus.

**4. Interaksi Keyboard (Keyboard Support)**
Kalkulator dapat digunakan langsung dengan keyboard, mirip seperti kalkulator sistem operasi.
