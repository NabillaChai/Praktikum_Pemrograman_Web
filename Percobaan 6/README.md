# Weather Dashboard

Weather Dashboard adalah aplikasi web sederhana yang menampilkan informasi cuaca secara real-time dengan tampilan yang rapi dan mudah digunakan. Aplikasi ini menampilkan suhu, kondisi cuaca, angin, serta prakiraan beberapa hari ke depan. Dilengkapi fitur pencarian kota, favorit, mode gelap/terang, dan pembaruan otomatis, aplikasi ini nyaman digunakan di berbagai perangkat.

<br>

## Fitur Utama

* **Current Weather** – Suhu, kelembaban, kecepatan angin, dan kondisi cuaca terkini.
* **5-Day Forecast** – Prediksi cuaca 5 hari ke depan dengan suhu minimum dan maksimum.
* **Search + Autocomplete** – Pencarian kota dengan saran otomatis.
* **Favorite Cities** – Simpan hingga 10 kota favorit.
* **Unit Toggle** – Pilih Celsius (°C) atau Fahrenheit (°F).
* **Theme Toggle** – Mode terang dan gelap.
* **Auto Refresh** – Pembaruan data otomatis setiap 5 menit.
* **Manual Refresh** – Tombol untuk memperbarui data secara manual.
* **Geolocation** – Deteksi lokasi otomatis saat aplikasi pertama kali dibuka.
* **Responsive Design** – Tampilan optimal di mobile, tablet, dan desktop.
